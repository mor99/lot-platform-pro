# Lot Platform Pro

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```


### 开始项目

```bash
npm start
```

or

```bash
yarn start
```

### 打包项目

```bash
npm run build
```

or

```bash
yarn build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### 测试

```bash
npm test
```
